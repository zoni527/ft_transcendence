# Backend endpoints
## Active endpoints
### GET
| URI               | DESCRIPTION                   |
| ---               | ---                           |
| /api/users        | Get JSON list of all users    |
| /api/users/:id    | Get user identified by id     |
| /api/recipes      | Get JSON list of all recipes  |
| /api/recipes/:id  | Get recipe identified by id   |
### POST
| URI               | DESCRIPTION                   |
| ---               | ---                           |
| /api/users        | Create new user               |
| /api/recipes      | Create new recipe             |
### PATCH
| URI               | DESCRIPTION                   |
| ---               | ---                           |
| /api/users        | Modify existing user          |
## Future endpoints
### DELETE
| URI               | DESCRIPTION                       |
| ---               | ---                               |
| /api/users/:id    | Delete user identified by id      |
| /api/recipes/:id  | Delete recipe identified by id    |
